Task 1: Increase all customers’ age by 1
Goal: increment every customer’s age by 1, and reset to original before each timed run.

1. Increase ages
use shopdb

db.customers.updateMany(
  {},
  { $inc: { age: 1 } }
)

2.Reset before next run (example)

Age stored in a backup field like original_age:
db.customers.updateMany(
  {},
  { $set: { age: "$original_age" } }
)

Task 4: Refunded orders in the last 30 days (2025‑04‑02 to 2025‑05‑01)
Goal: using the fixed reference date 2025‑05‑01, treat “last 30 days” as the inclusive range 2025‑04‑02 through 2025‑05‑01.


I interpret ‘last 30 days’ as the inclusive date range from 2025‑04‑02 to 2025‑05‑01. Since order_date is stored as a string in YYYY‑MM‑DD format, standard $gte and $lte comparisons safely handle this range without needing to convert to ISODate

db.orders.find({
  order_date: {
    $gte: "2025-04-02",
    $lte: "2025-05-01"
  },
  payment_status: "refunded"
})

Task 5: Count orders by payment status
Goal: use aggregation to count orders that have payment_status equal to "paid", "pending", or "refunded".

db.orders.aggregate([
  {
    $match: {
      payment_status: {
        $in: ["paid", "pending", "refunded"]
      }
    }
  },
  {
    $group: {
      _id: "$payment_status",
      count: { $sum: 1 }
    }
  }
])


Task 6: Identify and mark premium customers
Goal: mark customers as premium if they have more than 5 orders or total spending greater than 1000.

Aggregate orders by customer
use shopdb

const prem_id = db.orders.aggregate([
{ $group: {
_id: "$customer_id",
total_orders: { $sum: 1 },
total_spending: { $sum: "$total_amount" }
}
},
{ $match: {
$or: [
{ total_orders: { $gt: 5 } },
{ total_spending: { $gt: 1000 } }
]
}
} ]).toArray().map(d => d._id);

Update customers as premium
db.customers.updateMany(
{ _id: { $in: prem_id } },
{ $set: { is_premium: true } }
)

Task 8: Find orders containing a specific product
Goal: return orders that include a given product_id and show only matching items.

Example product reference
db.products.findOne();

Filter orders by product
db.orders.aggregate([
{ $match: { "items.product_id": 1 } },
{ $project: {
_id: 1,
customer_id: 1,
items: {
$filter: {
input: "$items",
as: "item",
cond: { $eq: ["$$item.product_id", 1] }
}
}
}
}
])

Goal: delete customers who do not appear in the orders collection.

Get customer IDs from orders
const ids = db.orders.distinct("customer_id");

Delete non-matching customers
db.customers.deleteMany(
{ _id: { $nin: ids } }
)

Task 10: Find categories purchased by young customers
Goal: identify distinct product categories purchased by customers under age 25.

Get young customer IDs
const youngids = db.customers.find(
{ age: { $lt: 25 } }
).toArray().map(d => d._id);

Aggregate categories from their orders
db.orders.aggregate([
{ $match: { customer_id: { $in: youngids } } },
{ $unwind: "$items" },
{ $group: {
_id: "$items.category"
}
}
])
